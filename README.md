## Course Overview

Advanced developers are expected to master database selections, and this course provides an excellent comparison of different types of databases that Python can interact with‚ÄîSQLite, MySQL, and PostgreSQL. Taught by Kathryn Hodge, this course takes you through the essentials of setting up, querying, and optimizing databases in Python.


### Course Modules

#### Introduction to Databases in Python
- Database definition and types
- Python Database API

- Creating SQLite database
- Adding multiple records
- Filtering records

- SQL expression language
- Chapter Challenge

#### Using MySQL in Python
- Creating MySQL database
- Building Tables
- Inserting Data
